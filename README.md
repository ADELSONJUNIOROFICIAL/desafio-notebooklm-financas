# 💰 NotebookLM: A Psicologia Financeira e o Planejamento Prático

Este repositório é o resultado do desafio prático da DIO: "Explorando o NotebookLM para Aprendizagem Ativa". O projeto foca na análise profunda da obra **"A Psicologia Financeira"** de Morgan Housel, integrando conceitos comportamentais com as diretrizes técnicas de mercado para 2026.

---

## 🎯 Objetivo do Projeto

- Analisar como o comportamento e o ego influenciam as decisões financeiras.
- Estudar as lições de Morgan Housel sobre a construção de riqueza duradoura.
- Aplicar conceitos de **Margem de Segurança** e **Juros Compostos** no cotidiano.
- Integrar a filosofia financeira com roteiros técnicos de planejamento orçamentário.
- Criar um guia de consulta rápida para o desenvolvimento de uma mentalidade de abundância.

## 🧠 Engenharia de Prompts e "Cicatrizes"

Para extrair o melhor do NotebookLM, utilizei as seguintes estratégias:

* **Prompt de Cruzamento:** *"Combine a lição sobre 'Margem de Erro' do livro com a tabela de gastos do InfoMoney. Como devo ajustar meu fundo de emergência se o mercado estiver volátil?"*
* **Prompt de Personificação:** *"Atue como um educador financeiro e crie um resumo para um jovem que ganha 2 salários mínimos, focando apenas na lição 'Riqueza é o que você não vê'."*
* **Desafios Encontrados (Troubleshooting):** Inicialmente, a IA misturou conceitos de 2023 com os de 2026. Tive que refinar o prompt instruindo-a a: *"Priorizar os dados do Guia InfoMoney 2026 para prazos e números, mas usar o tom filosófico de Morgan Housel para os conselhos"*. Isso garantiu a precisão técnica e a profundidade moral do texto.

---

## 🧮 Principais Conceitos Abordados

✔ **Psicologia do Dinheiro:** O sucesso financeiro é 80% comportamento e apenas 20% conhecimento técnico.
✔ **Margem de Segurança:** A importância de ter um espaço para o erro no seu planejamento para sobreviver aos imprevistos.
✔ **Riqueza vs. Estilo de Vida:** Entender que riqueza é o ativo acumulado (invisível) e não o gasto ostensivo (visível).
✔ **Liberdade:** O maior dividendo que o dinheiro paga é o controle sobre o seu próprio tempo.
